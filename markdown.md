## Link

string은 보여지는 부분, url은 연결할 곳을 나타냄

`[string](url)`

- 다른 페이지로 이동하는 링크를 삽입

- 필요하다면 파일 경로 넣어 다운로드 가능한 링크로 만들 수 있음

## Image

링크와 비슷하지만 이미지를 삽입

`[sting](img_url)`

- 이미지의 너비와 높이는 조절할 수 없음

- 조절이 필요하다면 HTML 사용
